# Build Week PRD
**User:** EU SME product/security team. **Problem:** evidence exists across files and scans but loses traceability as products change. **Outcome:** one verifiable path from source to approved claim, stale impact, incident draft, and dossier manifest. **Non-goals:** legal advice, compliance certification, scanning, or external submission.

Acceptance: grounded source span; reviewer action and audit metadata; deterministic component diff; missing incident facts remain unknown; reproducible hashes; tenant isolation smoke test.
