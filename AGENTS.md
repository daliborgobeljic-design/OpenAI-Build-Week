# Engineering contract
- Scope: Build Week BW0–BW3 only. No push, PR, deploy, video publication, or submission without owner approval.
- Never claim CRA compliance or legal advice. AI outputs remain suggestions until human approval.
- Use synthetic data only; treat uploads as untrusted; enforce tenant scope server-side.
- Required checks: `npm run lint`, `npm test`, `npm run build`.
- Definition of done: implementation, proportionate tests, accessible states, auditability, docs, reviewed diff, known risks.
